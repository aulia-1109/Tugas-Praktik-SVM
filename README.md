# Tugas Praktik SVM

## Deskripsi
Repository ini berisi tugas praktik klasifikasi data menggunakan algoritma Support Vector Machine (SVM) pada Iris Dataset.

## Dataset
Sumber Dataset:
https://archive.ics.uci.edu/ml/datasets/iris

## Tahapan Pengerjaan
1. Preprocessing Data
   - Cek missing value
   - Scaling data menggunakan StandardScaler

2. Split Data
   - Train 80%
   - Test 20%

3. Modeling
   - Algoritma: Support Vector Machine (SVM)
   - Kernel: RBF

4. Evaluasi
   - Accuracy
   - Precision
   - Recall
   - F1-Score

5. Visualisasi
   - Distribusi kelas
   - Confusion Matrix

## Hasil
Model SVM dengan kernel RBF menghasilkan akurasi sekitar 96%.
